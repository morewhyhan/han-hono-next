# Han-Hono-Next

## 技术栈

- **前端**：Next.js 14 (App Router) + React 18
- **后端**：Hono  
- **数据库**：SQLite 
- **ORM**：Prisma
- **样式**：Tailwind CSS + Shadcn UI
- **编程语言**：TypeScript

## 快速开始：

```bash
### 克隆项目
git clone https://github.com/morewhyhan/han-hono-next.git
cd han-hono-next

### 安装依赖
npm install

### 数据库迁移
npx prisma migrate dev

### 启动开发服务器
npm run dev
```

访问: http://localhost:3000
